To see and use Flex Gateway Connected Mode capabilities, organizations must have a license.
The entitlement is provided by sales operations through an admin console.

Entitled organizations see a *Flex Gateway* tab in the Runtime Manager UI, and two new types of Runtime Manager permissions:

[%header,cols="20a,60a"]
|===
|Permission |Description
|Manage Gateways |Access to create, update, and delete Flex Gateway resources. A user with this permission will be able to add gateways, unregister/delete gateways, and add/edit configurations.
|Read Gateways|Access to view gateways, replicas, and configurations.
|===
