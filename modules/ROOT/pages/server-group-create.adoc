= Create a Server Group (Hybrid)
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]


. Sign in to the Anypoint Platform and click *Runtime Manager*.
. In the menu on the left, click *Servers*, then click *Create Group*.
. In the *Create Server Group* page, enter the name for the server group, select the servers to include in your new server group, and click *Create Group*.
+
[TIP]
The group name must not start or end with a dash, must be between 3-40 characters long, must be unique, and contain only letters, numbers, or dashes.

The new server group appears in the *Servers* list. Click the server group name to see the list of servers that belong to the group.
