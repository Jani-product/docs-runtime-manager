You can see the details of the logs of your Flex Gateway in Anypoint Monitoring.

The Runtime Manager Flex Gateway UI provides you with a shortcut to access Anypoint Monitoring:

. From Anypoint Platform, select *Runtime Manager*.
. Select *Flex Gateway* in the left menu.
. Click the name of the gateway to which you want to add an API.
. Click *Logs*.
. Click *View logs in Anypoint Monitoring*.
+
This action takes you to the Anypoint Monitoring UI.
