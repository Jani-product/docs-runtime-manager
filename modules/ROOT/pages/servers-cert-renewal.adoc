Register a Mule server in Runtime Manager requires a valid certificate provisioned and signed by Runtime Manager. The certificate ensures secure communications between Runtime Manager and the Runtime Manager agent. Before a certificate expires, you can renew it directly from Runtime Manager. You cannot renew a certificate after it expires. You can update a certificate for a specific server or multiple servers within a cluster or group.

The expiration date of the certificate appears in the server dashboard. 

To renew a certificate for a server:

. From Anypoint Platform, select *Runtime Manager*.
. Click the *Servers* tab.
. Verify that the server status is *Running*.
+
The server must be running to renew the certificate.
. Click the server name.
. Click *Settings*.
. From the *Actions* menu, select *Renew Certificate*.
. Click the checkbox to confirm your choice, and then click *Renew*.
. Check *Certificate expiration date* to confirm that the certificate was successfully renewed:
+
.The arrow shows the *Certificate expiration date* on the *Settings* page.
