You might want to remove a server from one group and add it to another group.

To remove a server from a group:

. From Anypoint Platform, select *Runtime Manager*.
. In Runtime Manager, click *Servers* in the left menu.
. Click the type to display the server group details pane on the right.
. In the details pane, click the *X* icon to the right of the server to remove:
+
image::removeserverfromagroup.png[RemoveServerFromAGroup]

. In the *Remove* confirmation popup, select the checkbox to verify that you want to remove the server from the group, and then click *Remove*.
