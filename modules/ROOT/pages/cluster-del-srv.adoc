You can remove a server from a cluster without affecting the server or cluster.


To remove a server from a cluster:

. From Anypoint Platform, select *Runtime Manager*.
. Select *Servers* in the left menu.
. Click *Cluster* in the *Type* column to display the details pane.
. In the details pane, click the *X* icon to the right of the server to remove:
+
.The arrow shows the *X* icon in the details pane.
image::rtm-remove-server-from-cluster.png[X icon]

. In the *Remove* confirmation popup, select the checkbox to verify that you want to remove the server from the cluster, and then click *Remove*.
