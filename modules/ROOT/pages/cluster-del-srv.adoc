= Remove a Server From a Cluster (Hybrid)
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

You can remove a server from a cluster without affecting the server or cluster.

To remove a server from a cluster:

. In Runtime Manager, click the *Servers* tab.
. Click *Cluster* in the *Type* column for the cluster, then click *Manage Cluster* in the right pane.
. In the cluster settings page, click the *Servers* tab.
. Click the server name link, then click *Remove Server from Cluster* in the top right.
+
image::cluster-remove-server-from.png[]

. At the prompt, click the checkbox to confirm, then click *Remove*.
