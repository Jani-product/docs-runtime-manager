To set up a free developer Salesforce account:

== Register

Go to http://developer.salesforce.com/signup and register.

== Reset Your Security Token

To reset:

. After logging in to Salesforce, go to *Setup*.
. Select your name from the banner at the top of the window, and select *My Settings*.
. Select *Personal*.
. Select *Reset My Security Token* and press the *Reset Security Token* button.

An email is sent to you with your new security token.

Salesforce uses password plus security token for API authentication.
