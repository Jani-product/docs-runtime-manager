Use the server *Settings* page to configure server properties and view alerts. You can also perform other actions from this page, such as delete a server group.  

To access the server *Settings* page:

. From Anypoint Platform, select *Runtime Manager*.
. Click the *Servers* tab.
. Either: 
+
** Click the type on the *Servers page*, and then click *Manage Server*, *Manage Group*, or *Manage Cluster* button on the details pane.
+
image::managing-servers-ebc52.png[]
** Click the name of the server, server group, or cluster on the *Servers* page, then click *Settings*.
+
image::managing-servers-baa89.png[]
