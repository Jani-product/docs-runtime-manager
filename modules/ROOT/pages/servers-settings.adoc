= Access Server Settings (Hybrid)

include::_attributes.adoc[]
endif::[]

The *Settings* page can be accessed by either:

* Clicking the *Manage Server*, *Manage Group*, or *Manage Cluster* button on the right panel that opens when you select a server.

+
image::managing-servers-ebc52.png[]

* Directly clicking on the name of the server, group, or cluster on the table.

+
image::managing-servers-baa89.png[]

The *Settings* page contains the same information that's accessible on the right panel of the *Servers* section, with the addition of an *Alerts History* tab, that displays a record of the triggered alerts on this server.
